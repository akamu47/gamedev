This project teaches you about Progressive Web Apps (PWAs) ‚Äî the same technology used by companies like Twitter and Starbucks for their mobile apps. You'll also learn about touch controls, responsive design (making things work on any screen size), and the Canvas API for drawing game graphics.

üí° What's a PWA (Progressive Web App)?

A PWA is a website that can be "installed" on a phone and acts just like a real app ‚Äî it has an icon on the home screen, works full-screen, and can even work offline. The best part? You don't need to pay Apple or Google to put it in their app stores. Just share a link and your friends can install it instantly!

üß† What You'll Learn Along The Way

Canvas API ‚Äî How to draw graphics with code. Every shape, animation, and particle effect is drawn pixel by pixel. This is the foundation of 2D game development.

Touch Events & Mobile Development ‚Äî How phones detect taps, swipes, and drags. You'll learn to make interfaces that feel native and responsive on any device.

Progressive Web Apps ‚Äî How to turn a website into an installable app. This is a real professional skill ‚Äî many companies use PWAs to reach mobile users without going through app stores.

Game Design Thinking ‚Äî By inventing your own game, you'll learn what makes games fun: challenge curves, risk vs reward, feedback loops, and the "just one more try" feeling.

Product Thinking ‚Äî By sharing your app and collecting feedback, you'll learn how real developers think about users. What do people actually enjoy? What's confusing? This is how every app you use gets better over time.
